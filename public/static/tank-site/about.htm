Despre program

Tank Action este un joc realizat de mine in clasa a IX-a, utilizand limbajul Borland Pascal 7.0.

Tank Action este primul meu program de anvergura. Continand peste 1200 de linii de cod, structurat si cuprinzand diversi algoritmi dezvoltati de mine precum si utilizand diverse tipuri de date alocate static si dinamic programul a fost catalogat drept o realizare exceptionala pentru varsta respectiva de catre fostul meu profesor de informatica de la acea vreme, Catalin Francu, olimpic international la informatica si actualmente doctorand la M.I.T.

Desi poate nu la fel de impresionant ca alte programe datorita interefetei DOS, acesta nu numai ca are peste 1200 de linii de program dar contine si solutii informatice, algoritmi, structuri si blocuri logice complexe dezvoltate empiric si nu preluate ca atare din diverse surse.

Puncte importante ale programului :

  • Folosirea fisiere cu tip si fara tip ;
  • Alocarea dinamica a memoriei ;
  • Emularea pe vectori, matrice, record-uri, proceduri si functii a programarii orientate pe obiect ;
  • Utilizarea capacitatile grafice oferite de limbaj ;
  • Dezvoltarea de algoritmi de miscare semi-haotica a adversarilor, miscari cu componenta fixa si aleatoare, cu ocolire de obstacole si testarea oportunitatii de atac ;
  • Utilizare unitatii DOS pentru captarea si prelucrarea timpului ;
  • Sortare in fisier cu tip ;
  • Utilizarea mouse-ului in mediu DOS, cu initializare, etc. ;
  • Folosirea procedurilor de lucru cu imagini GetImage si PutImage precum si a tehnicii de animatie prin rescriere in modul XORPUT ;
  • Aplicatii ale functiilor trigonometrice in generarea sinusoidelor si a artificiilor;
  • Editor simplist de text in mod grafic cu optiunea de stergere.
  • Etc.
